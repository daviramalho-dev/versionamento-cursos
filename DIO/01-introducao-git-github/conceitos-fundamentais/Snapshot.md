Snapshot = Fotografia do seu projeto em um momento específico.

Quando você faz [[Commit]]:
- Git captura o estado EXATO de todos os arquivos
- Salva uma referência para recuperar depois
- Você pode voltar a qualquer snapshot anterior

Exemplo:
- Snapshot 1: Projeto com index.js (v1)
- Snapshot 2: Projeto com index.js (v2) + app.js novo
- Snapshot 3: Projeto com tudo editado

Cada snapshot tem um [[Sha-1]] único!
